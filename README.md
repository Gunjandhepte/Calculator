# Calculator Project (C++)

This is a simple Calculator project developed using C++ and Object Oriented Programming concepts.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Division by zero handling
- Menu driven program

## Technologies Used
- C++
- OOP Concepts

## How to Run
1. Open terminal or command prompt
2. Compile the program:
   g++ calculator.cpp -o calculator
3. Run the program:
   ./calculator
